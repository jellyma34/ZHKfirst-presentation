# Уменьшение масштаба генплана

## ✅ Изменения выполнены

### 🎯 Цель
Уменьшить визуальный масштаб генплана на 15-20% для более аккуратного отображения и гармоничного баланса с панелью фильтров.

---

## 📊 Что было изменено

### Масштаб уменьшен на 20%

**Было:**
```jsx
transform: 'scale(0.65)'  // 65%
```

**Стало:**
```jsx
transform: 'scale(0.52)'  // 52% (уменьшение на 20%)
```

**Местоположение:** `src/pages/Landscaping.jsx`, строка 47

## 📐 Сохраненные параметры

### Не изменено:

✅ **transformOrigin:** `'center top'` - центрирование от верха  
✅ **transition:** `'transform 0.3s ease'` - плавная анимация  
✅ **alignSelf:** `'flex-start'` - выравнивание по верху  
✅ **maxWidth:** `'1400px'` - максимальная ширина  
✅ **background:** `'transparent'` - без фона  
✅ **border:** `'none'` - без рамки  
✅ **boxShadow:** `'none'` - без теней  

### Стили изображения:

✅ **width:** `'100%'` - адаптивная ширина  
✅ **height:** `'auto'` - пропорциональная высота  
✅ **objectFit:** `'contain'` - без обрезки  
✅ **borderRadius:** `0` - без скругления  

---

## 📱 Адаптивность сохранена

Масштаб 0.52 применяется на всех экранах Desktop.

На мобильных и планшетах CSS автоматически переключает на scale(1):

```css
@media (max-width: 1024px) {
  .landscape-plan-container {
    transform: scale(1) !important;
  }
}
```
