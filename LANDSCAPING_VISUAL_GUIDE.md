## 🔄 Работа со слоями

### Базовое состояние (нет активных слоёв):

```
┌──────────────────────┐
│                      │
│   ОСНОВНОЕ           │
│   ИЗОБРАЖЕНИЕ        │
│   ГЕНПЛАНА           │
│                      │
│   z-index: 1         │
│                      │
└──────────────────────┘
```

### С активными слоями:

```
        ┌─────────────┐  ← Зоны отдыха (z:4)
       ┌┴────────────┐│
      ┌┴────────────┐││  ← Пешеходные (z:3)
     ┌┴────────────┐│││
    ┌┴────────────┐││││  ← Озеленение (z:2)
    │ ОСНОВНОЕ    ││││
    │ ИЗОБРАЖЕНИЕ │││││  ← Основа (z:1)
    │             ││││
    └─────────────┘│││
     └─────────────┘││
      └─────────────┘│
       └─────────────┘

Все слои:
- position: absolute
- top: 50%, left: 50%
- transform: translate(-50%, -50%)
- opacity: 0.85
```

## 🛠️ Технические детали

### Z-index иерархия:
```
z-index: 6  ← Зоны отдыха (если активен)
z-index: 5  ← Пешеходные зоны
z-index: 4  ← Парковки
z-index: 3  ← Освещение
z-index: 2  ← Детские площадки
z-index: 2  ← Озеленение
z-index: 1  ← Основное изображение
```

### Transform стратегия:
```
Контейнер: scale(0.65)
     │
     └──→ Изображение: 100% (относительно контейнера)
              │
              └──→ Слои: 100% (синхронизированы)
```
