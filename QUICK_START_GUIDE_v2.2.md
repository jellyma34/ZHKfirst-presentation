## 🎯 Пошаговая инструкция

### 1. Открытие редактора

```
Планировочные решения 
  → Движение пешеходов ✓ (активировать)
  → Редактировать маршруты 🔘
```

### 2. Создание маршрута

```
1. Кликните по плану → появится точка 🟡
2. Кликните еще → линия соединит точки 🟡──🟡
3. Добавьте больше точек → 🟡──🟡──🟡──🟡
4. Перетащите точку мышью для коррекции ↔️
```

### 3. Завершение маршрута

```
Панель управления (под планом):

┌────────┐ ┌──────────┐ ┌────────────┐
│Текущий │ │ Маршруты │ │  Действия  │
├────────┤ └──────────┘ ├────────────┤
│Точек: 5│              │[Сохранить] │
│────────│              │[Очистить]  │
│[Отмена]│              │[Закрыть]   │
│[Готово]│              └────────────┘
└────────┘

Нажмите: [Готово] ✓
```

### 4. Сохранение

```
[Сохранить] 💾
  ↓
✅ Сохранено в localStorage
✅ Скачан файл routes-[этаж].json
```

## 🎨 Цвета маршрутов

- **🟡 Жёлтый (пунктир)** - текущий редактируемый
- **🔵 Голубой** - сохранённые маршруты

---

## ⌨️ Действия панели

### Блок "Текущий маршрут":
- **Отмена** 🔙 - удалить последнюю точку
- **Завершить** ✅ - сохранить маршрут в список

### Блок "Маршруты":
- Список всех сохранённых маршрутов
- **🗑️** - удалить конкретный маршрут

### Блок "Действия":
- **Сохранить** 💾 - в localStorage + скачать JSON
- **Очистить всё** 🗑️ - удалить все маршруты
- **Закрыть** ✖️ - выйти из редактора

---

## 💡 Подсказки

### ✅ Правильно:
- Кликайте по основным путям (коридоры, холлы)
- Создавайте 2-3 маршрута минимум
- Сохраняйте после редактирования
- Точки можно перетаскивать

### ❌ Ошибки:
- Не создавайте маршруты с 1 точкой (минимум 2)
- Не забывайте нажать "Завершить"
- Не забывайте "Сохранить" перед закрытием

## 🐛 Если что-то не работает

### Проблема: Не вижу панель управления
**Решение:** Прокрутите страницу вниз, панель под планом

### Проблема: Не могу кликнуть по плану
**Решение:** Убедитесь, что редактор открыт (кнопка активна)

### Проблема: Точки не создаются
**Решение:** Кликайте по самому плану, не по краям

### Проблема: Не могу завершить маршрут
**Решение:** Нужно минимум 2 точки для завершения
