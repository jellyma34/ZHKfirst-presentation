# Маршруты пешеходов

Эта директория содержит JSON-файлы с настраиваемыми маршрутами движения пешеходов для разных планов.

## Структура файла маршрута

```json
[
  {
    "id": "route-1",
    "points": [
      { "x": 120, "y": 170 },
      { "x": 250, "y": 170 },
      { "x": 420, "y": 200 }
    ]
  },
  {
    "id": "route-2",
    "points": [
      { "x": 200, "y": 400 },
      { "x": 400, "y": 350 },
      { "x": 950, "y": 400 }
    ]
  }
]
```

## Именование файлов

Файлы должны называться по ID плана:
- `genplan.json` - для генплана (Благоустройство)
- `underground-ground.json` - для подземно-надземного этажа
- `general-floor-1.json` - для первого этажа
- `section1-floor-2.json` - для секции 1, этаж 2
- и т.д.

## Использование

1. Откройте редактор маршрутов через кнопку "Редактировать маршруты"
2. Кликайте по плану, чтобы создать точки маршрута
3. Перетаскивайте точки для коррекции
4. Нажмите "Завершить" для сохранения маршрута
5. Нажмите "Сохранить" для экспорта JSON файла
6. Файл автоматически скачается и будет сохранен в localStorage

## Загрузка маршрутов

Маршруты загружаются в следующем порядке:
1. Из localStorage (приоритет)
2. Из файла в `/routes/[planId].json`
3. Если ничего не найдено - используются дефолтные маршруты
