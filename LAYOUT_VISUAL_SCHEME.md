## 🎭 Слои наложения (Stack)

```
Z-index структура:

   ┌──────────────────┐  z-index: 4 (Слой 3 - 2-комнатные)
   │ ┌──────────────┐ │
   │ │ ┌──────────┐ │ │  z-index: 3 (Слой 2 - 1-комнатные)
   │ │ │          │ │ │
   │ │ │  ОСНОВА  │ │ │  z-index: 1 (Основное изображение)
   │ │ │          │ │ │
   │ │ └──────────┘ │ │
   │ └──────────────┘ │
   └──────────────────┘

Все слои:
- position: absolute
- top: 50%, left: 50%
- transform: translate(-50%, -50%)
- maxWidth: 1200px (синхронизировано)
```

## 🔍 Zoom взаимодействие

```
Контейнер (transform: scale(1.8))
    │
    ├──→ Изображение (transform: scale(zoomLevel/100))
    │       │
    │       ├──→ 100% (по умолчанию)
    │       ├──→ 125% (zoom +)
    │       ├──→ 150% (zoom +)
    │       └──→ 200% (max)
    │
    └──→ Слои (transform: translate(-50%,-50%) scale(zoomLevel/100))
            │
            └──→ Синхронизированы с изображением

Итоговый масштаб = 1.8 (контейнер) × zoom (пользователь)
Пример: 1.8 × 1.5 = 2.7x общее увеличение
```
