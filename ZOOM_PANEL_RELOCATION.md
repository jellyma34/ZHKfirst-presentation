# 🎛️ Перемещение панели масштаба под план

## ✅ Выполненные изменения

### 1. 📍 Панель масштаба перенесена под план

**Файл**: `src/pages/PlanningSolutions.jsx`

#### Структура ДО изменений
```jsx
<div className="planning-layout">
  <LeftPanel />        {/* Выбор этажа */}
  
  <CenterPanel>        {/* План */}
    <PlanImage />
  </CenterPanel>
  
  <RightPanel />       {/* Фильтры слоёв */}
</div>

{/* Панель масштаба ВНИЗУ страницы */}
<ZoomControls />
```

#### Структура ПОСЛЕ изменений
```jsx
<div className="planning-layout">
  <LeftPanel />        {/* Выбор этажа */}
  
  <CenterPanel>        {/* План + Панель масштаба */}
    <PlanImage />
    <ZoomControls />   {/* ✅ ПЕРЕНЕСЕНО СЮДА */}
  </CenterPanel>
  
  <RightPanel />       {/* Фильтры слоёв */}
</div>
```

**Результат**: Панель управления масштабом теперь находится **под планом**, а не внизу страницы

---

### 2. 🎨 Обновлена структура центрального блока

**Изменено:**
```jsx
// Было:
<motion.div
  className="floor-plan-container"
  style={{
    display: 'flex',
    justifyContent: 'center',
    alignItems: 'flex-start',
    width: '100%',
    alignSelf: 'start'
  }}
>

// Стало:
<motion.div
  className="floor-plan-container plan-wrapper-column"
  style={{
    display: 'flex',
    flexDirection: 'column',      // ✅ колоночное расположение
    justifyContent: 'center',
    alignItems: 'center',
    width: '100%',
    alignSelf: 'start',
    gap: '20px'                   // ✅ отступ между планом и панелью
  }}
>
```

## 🔧 Технические детали

### Flex Direction
```css
/* Центральный блок теперь вертикальный: */
flex-direction: column;

/* Элементы располагаются друг под другом: */
┌───────────┐
│   План    │
├───────────┤  ← gap: 20px
│ Масштаб   │
└───────────┘
```

### Gap между элементами
```css
gap: 20px;
/* Создает отступ между планом и панелью масштаба */
```

### Z-index
```css
.zoom-controls {
  z-index: 5;
  /* Панель всегда поверх других элементов */
}
```
