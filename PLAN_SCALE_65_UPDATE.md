### 3️⃣ Сохранена плавная масштабировка

**Контейнер:**
```jsx
transform: `scale(${zoomLevel / 100})`
transition: 'transform 0.3s ease'
```

**Как работает:**
- Начальное значение: **65%** (zoomLevel = 65)
- Через слайдер: от **25%** до **200%**
- Плавная анимация **0.3s**
- Transform origin: **center top**

**Функция сброса:**
```jsx
const resetZoom = () => {
  setZoomLevel(65);  // Возврат к 65%, было 100%
};
```

## 🚀 Как работает масштабирование

### Контролы зума:

**Слайдер:**
```
25% ←────●────────────────→ 200%
        65% (по умолчанию)
```

**Кнопки:**
- **[-]** Zoom Out: уменьшает на 25%
- **[+]** Zoom In: увеличивает на 25%
- **[↻]** Reset: возврат к **65%**

**Формула:**
```
transform: scale(zoomLevel / 100)

Примеры:
- zoomLevel = 25  → scale(0.25) = 25%
- zoomLevel = 65  → scale(0.65) = 65% ← по умолчанию
- zoomLevel = 100 → scale(1.0)  = 100%
- zoomLevel = 200 → scale(2.0)  = 200%
```
