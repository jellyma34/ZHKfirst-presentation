# 🎯 Быстрая инструкция v2.3 - Панель в сайдбаре

## ✨ Что нового?

Панель управления редактором маршрутов теперь **в боковой панели "Фильтры слоёв"**!

## 📍 Новое расположение

```
┌────────────────────────────────────────┐
│ [Выбор]      ПЛАН          [Фильтры]  │
│                            │           │
│                            │ Фильтры   │
│  План                      │ слоёв     │
│  полностью                 ├───────────┤
│  свободен                  │✏️ Toggle  │
│  для                       ├───────────┤
│  рисования                 │ Панель    │ ← ВОТ ЗДЕСЬ!
│                            │ управл-я  │
│     🔍 Масштаб             │           │
└────────────────────────────────────────┘
```

---

## 🎯 Пошаговая инструкция

### 1. Активация редактора

```
1. Движение пешеходов ✓
2. Кнопка "Редактировать маршруты" появилась
3. НАЖАТЬ на кнопку
   ↓
4. Кнопка стала ЗЕЛЁНОЙ "✓ Редактирование активно"
5. Под кнопкой появилась панель управления
```

### 2. Кнопка-переключатель

**Неактивна (серо-синяя):**
```
┌──────────────────────────┐
│ ✏️ Редактировать        │ ← Кликните
│    маршруты              │
└──────────────────────────┘
```

**Активна (зелёная):**
```
┌──────────────────────────┐
│ ✓ Редактирование        │ ← Кликните снова
│   активно                │   для закрытия
└──────────────────────────┘
```

### 3. Панель управления

```
В правой панели под кнопкой:

┌─────────────────────────┐
│ ✏️ Подсказка            │
├─────────────────────────┤
│ Текущий маршрут         │
│ Точек: 3                │
│ [Отменить] [Завершить]  │
├─────────────────────────┤
│ Маршруты (2)            │
│ • Маршрут 1 [🗑️]        │
│ • Маршрут 2 [🗑️]        │
├─────────────────────────┤
│ [💾 Сохранить]          │
│ [🗑️ Очистить]          │
│ [✖ Закрыть]             │
└─────────────────────────┘
```

---

## 🎨 Цвета

- **Подсказка** - светло-серая
- **Текущий маршрут** - жёлтая рамка
- **Сохранённые** - голубая рамка
- **Кнопки:**
  - ↩ Отменить - жёлтая
  - ✓ Завершить - зелёная
  - 💾 Сохранить - синяя
  - 🗑️ Очистить - красная
  - ✖ Закрыть - серая

---

## ⚡ Быстрый старт

```
1. Движение пешеходов ✓
2. Редактировать маршруты → зелёная
3. Кликайте по плану 🟡🟡🟡
4. Завершить (в панели справа)
5. Сохранить (в панели справа)
6. Закрыть (в панели справа)
```

---

## ✅ Преимущества

1. ✅ **План полностью свободен** - никаких элементов поверх
2. ✅ **Всё в одном месте** - все кнопки в правой панели
3. ✅ **Toggle-кнопка** - визуально видно, активен редактор или нет
4. ✅ **Компактно** - вертикальная компоновка экономит место

## 💡 Советы

### Для создания маршрута:
1. Кликайте по основным путям (коридоры)
2. Минимум 2 точки
3. Нажмите "Завершить" в панели
4. Создайте ещё 1-2 маршрута
5. Нажмите "Сохранить"

### Для редактирования:
- Перетаскивайте точки мышью
- Используйте "Отменить" для удаления последней точки
- Кнопка "Очистить" удаляет ВСЁ (будет подтверждение)

---

## ❌ Частые ошибки

### Ошибка: Не вижу панель управления
**Решение:** Проверьте правую панель, прокрутите вниз

### Ошибка: Кнопка не меняет цвет
**Решение:** Убедитесь, что "Движение пешеходов" активировано

### Ошибка: Не могу закрыть редактор
**Решение:** Нажмите зелёную кнопку ещё раз ИЛИ "Закрыть" в панели
