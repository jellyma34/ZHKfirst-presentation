# 🎯 Кнопки навигации — обновлено

## ✅ Что изменено

Кнопки переключения слайдов (❮ / ❯) теперь расположены **за пределами контейнера** на desktop, не перекрывая карточки.

---

## 📐 Новое расположение

### Desktop (> 1024px)
```
      -90px                                        -90px
        ❮                                            ❯
           ┌─────────────────────────────────┐
           │     [Левая]  [Центр]  [Правая]  │
           └─────────────────────────────────┘
```

### Tablet (768px - 1024px)
```
    -70px                                    -70px
      ❮                                        ❯
         ┌───────────────────────────┐
         │   [Левая]  [Центр]  [Правая]   │
         └───────────────────────────┘
```

### Mobile (< 768px)
```
   5px                                           5px
    ❮                                             ❯
      ┌───────────────────────────────────┐
      │          [Только центр]            │
      └───────────────────────────────────┘
```

---

## 🎨 Визуальные улучшения

### Размер
- **55x55px** (было 50x50px)
- Более удобный для клика

### Цвет и тени
```css
background: rgba(255, 255, 255, 0.95);
box-shadow: 0 4px 14px rgba(0, 0, 0, 0.25);
```

### Hover-эффект
```css
background: white;
transform: scale(1.15);
box-shadow: 0 6px 18px rgba(0, 0, 0, 0.3);
```
