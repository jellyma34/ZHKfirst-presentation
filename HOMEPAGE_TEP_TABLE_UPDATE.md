### 2. **Условный рендер таблицы**

Добавлена логика проверки `isTable`:

```jsx
{slide.isTable ? (
  // Таблица технико-экономических показателей
  <table className="tep-table">
    <thead>
      <tr>
        {slide.tableData.headers.map((header, i) => (
          <th key={i}>{header}</th>
        ))}
      </tr>
    </thead>
    <tbody>
      {slide.tableData.rows.map((row, rowIdx) => (
        <tr key={rowIdx}>
          {row.map((cell, cellIdx) => (
            <td key={cellIdx}>{cell}</td>
          ))}
        </tr>
      ))}
    </tbody>
  </table>
) : (
  // Обычный контент (для остальных карточек)
  Array.isArray(slide.subtitle)
    ? slide.subtitle.map((line, i) => (
        <p key={i} className="card-subtitle">{line}</p>
      ))
    : slide.subtitle && <p className="card-subtitle">{slide.subtitle}</p>
)}
```
