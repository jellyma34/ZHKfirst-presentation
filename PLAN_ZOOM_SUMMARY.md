# ✅ Увеличение плана в разделе "Планировочные решения"

## Что изменено

### 1. 🔍 Масштаб плана увеличен в 2 раза

```javascript
// src/pages/PlanningSolutions.jsx

const [zoomLevel, setZoomLevel] = useState(130); // было 65
```

**Расчет:**
- Было: 65% → `scale(0.65)`
- Стало: 130% → `scale(1.3)`
- **Увеличение: 130 ÷ 65 = 2 раза** ✅

---

### 2. 🔄 Кнопка сброса

```javascript
const resetZoom = () => {
  setZoomLevel(130); // было 65
};
```

✅ Кнопка "Сброс" возвращает к **130%** (новому базовому масштабу)
